xFid Encryption Application

Experimental console application expanding the functionality of a traditional trifid cipher from 3x3x3 to 5x5x5, with support for encrypting digits, specific casing, and a selection of punctuation and Latin characters. Allows for random cipher generation, saving created keys and messages, and encrypting through multiple passes of the same key. This allows the user to have 125! times the number of passes potential solutions to a single encryption. The goal is eventually to transfer it into a desktop or web application that will output encrypted save files.
